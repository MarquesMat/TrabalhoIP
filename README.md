# TrabalhoIP
O código foi escrito seguindo o API do Google Sheets. Procure a documentação para mais detalhes.

É necessário ter o arquivo credentials.json para efetuar o login na sua conta.

Na primeira vez que rodar o programa, será aberta uma janela no navegador para confirmar o login.

Após isto, será criado o arquivo token.json que permitirá o login imediato, mas sua funcionalidade é temporária.

Caso o programe pare de funcionar, apague o arquivo token.json e rode novamente.

Crie um arquivo ips_teste.txt com uma lista de IPs a serem testados.

Ao final do programa, será gerado o arquivo ips_livres.txt com os IPs que estão livres.
